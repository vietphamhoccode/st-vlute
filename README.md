# st-vlute
Ứng dụng dành cho sinh viên VLUTE theo dõi thời khoá biểu, bảng điểm, trạng thái học tập các môn học, giúp sinh viên theo dõi thời khoá biểu thay vì truy cập vào trang đào tạo để xem trong lúc app VLUTE ONLINE không hoạt động, app hoạt động độc lập không server chỉ dựa vào hoạt động api của hệ thống nhà trường sử dụng để lấy dữ liệu về và hiển thị
